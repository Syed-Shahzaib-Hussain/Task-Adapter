Redmine Java API 1.10 released: going JSON

Posted on May 17, 2012 by admin

We released Redmine Java API 1.10.0 today.
Significant changes:

  1. Exception handling changes are not backward compatible. See  https://github.com/taskadapter/redmine-java-api/wiki/Migration-Guide
  2. The library is now using JSON instead of XML when communicating with Redmine. Castor XML library dependency is dropped.

Project website: https://github.com/taskadapter/redmine-java-api
