Task Adapter 1.7.1: Important bugfix and new library for Microsoft Project (MPP, XML) files processing

Posted on April 2, 2012 by admin
Task Adapter 1.7.1 is ready for download.

Important bugfix: Issue relations are synchronized when tasks are updated in Remine.

Also a new version of MPXJ library is included into release (this is the library Task Adapter uses to read and write Microsoft Project files).

Tested with Redmine 1.3.1, Atlassian Jira 4.4.4, MantisBT 1.2.8, Microsoft Project 2010 Prof, current Github.com website.
