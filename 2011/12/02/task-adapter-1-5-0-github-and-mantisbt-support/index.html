Task Adapter 1.5.0: Github and MantisBT support

Posted on December 2, 2011 by admin

Task Adapter 1.5.0 is ready for download.
This is a major release, which supports 2 new systems: Github and MantisBT.
New features
  • Proxy support
  • Microsoft Project: added support for “priority” field.
Plus a lot of bugfixes and minor UI improvements. Full changes list. 
Tested with Redmine 1.2.2.devel.7885 , Atlassian Jira 4.4.3, MantisBT 1.2.8, Github.com website.
